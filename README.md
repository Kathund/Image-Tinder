# Image Tinder

Image Tinder is built using [**React**](https://react.dev/), [**Next.js**](https://nextjs.org/),
[**TypeScript**](https://typescriptlang.org/) and [**Tailwind CSS**](https://tailwindcss.com/) using [**t3 stack**](https://create.t3.gg/)

Page live on [**photos.kath.lol**](https://photos.kath.lol/)
